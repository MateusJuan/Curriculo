O Sistema de Escalas é um aplicativo para dispositivos móveis, seja ele IOS ou Android, e aplicação WEB que tem como objetivo facilitar a comunicação e organização de eventos e cultos para os membros da igreja. Neste sistema estou trabalhando sozinho, com o objetivo de criar experiências mais aprofundadas no desenvolvimento FrontEnd e BackEnd.
Utilizo React Native para o desenvolvimento do aplicativo, Node.js para o backend o qual está hospedado no Render e Supabase como banco de dados. O sistema está em desenvolvimento, porém já está com algumas pessoas o testado, estou utilizando do eas para gerar builds e atualizar o App.
